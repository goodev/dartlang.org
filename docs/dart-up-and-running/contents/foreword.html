Foreword

Lars Bak

Kasper Lund

当我们来到谷歌，WEB 浏览器发展已经超过六年了，这是一个令人着迷的世界，而且现在和以前大有不同。 目前互联网应用大量涌现， 但是其运行平台的性能不足以满足现状。这就我们设计和实现虚拟机的背景， 创建一个高性能的 JavaScript 引擎看起来是一个有趣的挑战。 是的，从头做起我们开发了 V8 JavaScript 引擎，并且在2008年把它内置到了Google Chrome中，我们非常自豪其带来的性能提升似乎影响了整个浏览器 产业。

即使新的性能使得 WEB 浏览器突破了不能运行大型的和复杂的互联网应用的限制，但是开发 大型的、需要高性能的 WEB 应用仍然很困难。 没有好的抽象化机制和清晰的语义，开发者经常受代码结构 复杂混乱的影响而走向失败。当代码基量越来越大这个问题自然越来越严重。 我们设计 Dart 编程语言来解 决这个问题，我们希望使用 Dart 的开发者将会更有创造力。

过去的一年里，我们阅读和编写了大量 的 Dart 代码，看到 Dart 开发者们创造出简洁优美的程序，我们非常高兴。这里有一些非常让人愉快的改 变，通过反射和添加类型声明向可维护的软件开发过程中增加了改变原型--这使得 Dart 不再像是小的实验， 而更像是一个大型项目。

Dart: 启动和运行 是一个实用的向导，它 介绍了 Dart 程序开发语言，并教你如何创建 Dart 应用程序。我们希望你喜欢 Dart 和这本书。
