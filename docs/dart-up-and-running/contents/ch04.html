总结

Dart 提供了一些工具来帮助你编写和部署 web 应用和命令行 应用：

pub: Dart 包 管理器

下载和安装包中的库。

Dart 编辑器

编辑、运行、调试 web 和 命令行 应用。

Dartium: 带有 Dart VM 的 Chromium 浏览器

运行 Dart web 应用。用带有 Dart VM 的 Chromium 运行 web 应用。

dart2js: Dart-to-JavaScript compiler

把你的 web 应用编译为 JavaScript 就可以在不包含 Dart VM 的浏览器中运行了。

dart:独立的 Dart VM

运行命令行应用、服务器端脚本或者其他不能在 浏览器中运行的 Dart 应用。

dartanalyzer: 静态 分析器

用命令行来分析 Dart 代码。

dartdoc: API 文档 生成器

从 Dart 代码中生成文档。

以上工具都包含在下载的 Dart 编辑器包中。你也可以单独 下载 Dartium，也可以单独 下载包含 pub、 dart2js、 dart、 dartanalyzer、 和 dartdoc 的 SDK。参考 下载界面 。

这些工具位于下载的 Dart 安装目录中的 dart-sdk/bin 目录中。 如果你想使用这些命令行工具，建议把 bin 目录添加到环境变量中去。
