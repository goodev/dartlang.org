dart: 独立的虚拟机(VM)

你可以用 dart 工具 (bin/dart) 来运行 Dart 命令行应用， 例如 服务器端脚本、代码或者服务器等。 在开发的时候，你也可以在 Dart 编辑器 中运行命令行应用。

基本用法

下面是在命令行运行 Dart 代码的一个示例：

lang-sh
dart test.dart

支持的参数

dart 支持如下一些常用的参数：

--enable-checked-mode 或者 --checked

启用 断言和类型检测 (checked mode).

--package-root=<path>

指定查找导入包的目录

--print-flags

打印出所有参数列表

启用检测模式

Dart 程序有两种运行模式： checked 或者 production （检测模式或者生产模式）。 默认情况下，Dart VM 使用的是生产模式。 我们推荐你用 检测模式 来开发和测试应用。

在检测模式下，赋值操作会进行类型检测，如果类型不匹配则会抛出 一个异常。而在生产模式下，类型定义没有任何效果。

断言也在检测模式下有用。一个 assert statement（断言语句） 检测一个布尔值， 如果该值为 false 则抛出一个异常。 同样，在生产模式下，断言也没任何效果。

使用 --checked 命令行参数可以启用 VM 的检测模式：

lang-sh
dart --checked test.dart
