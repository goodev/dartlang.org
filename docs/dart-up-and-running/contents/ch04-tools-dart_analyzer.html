dartanalyzer: 静态分析工具

你可以用 dartanalyzer 工具 (bin/dartanalyzer) 通过命令行来静态的分析 代码，检测 Dart 语言规范 中的一些错误和警告情况。

Note

Dart 编辑器 除了执行 dartanalyzer 功能外还有其他 一些常见错误检测。 之前版本的 dart_analyzer已经被淘汰了。

基础用法

下面是测试一个 Dart 代码的示例：

lang-sh
dartanalyzer --package-root code/packages test.dart

如 Table 4.1, “dartanalyzer 的退出码” 所示，dartanalyzer 的退出码 代表分析的结果。

Table 4.1. dartanalyzer 的退出码

退出码描述
0没有发现问题
1发现警告(但是没错误)
2发现错误

支持的参数

dartanalyzer 支持如下命令行参数：

--dart-sdk <path>

设置 Dart SDK 的目录。

--package-root <path>

设置使用 package: 导入的包的目录。

--show-package-warnings

不仅仅显示你应用中的 .dart 代码警告，同时还显示 所使用库（package: 导入的库）的警告信息。

--machine

生成可以分析的结果文件。

--version

显示分析器的版本。

--help

显示所有命令行参数。
