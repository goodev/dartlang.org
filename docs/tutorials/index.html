Dart 教程告诉你如何用 Dart 语言、工具和 API 创建 web 应用。

前提条件

  • 你已经知道如何编写结构化程序语言，例如 C 或者 Java。
  • 你熟悉面向对象编程概念。
  • 你可以不了解如何通过 DOM (Document Object Model)来实现浏览器编程。

出发吧！ 你可以按照从左到右的顺序 逐步学习本教程，也可以直接选择你需要的部分。
