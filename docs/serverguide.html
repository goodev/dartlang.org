Packages

  • http_server— Virtual directory, virtual host, and static file serving. (Dart team)

  • route— URL routing for the server (and the client). (Dart team)

  • args— Handle command-line arguments. (Dart team)

  • http— Connect to HTTP resources. (Dart team)

Find more packages useful for server-side and command-line apps at pub.dartlang.org.

Reference docs

  • dart:io— Includes support for UDP, TCP, HTTP client and server, files, directories, web sockets, processes, server sockets, and more.
  • args package— Help with handling command-line arguments.
  • dart:async— Asynchronous programming with Futures, Streams.
  • dart:convert— Converters for data types such as JSON and UTF-8.
