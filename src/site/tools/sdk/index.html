Dart 库和命令行工具

Dart SDK 包含了你需要开发 Dart 应用的 库和命令行工具。 要查看 Dart 开发可用的工具，请访问 see Dart工具页面。

下载 SDK

如果你使用了最新的 Dart 编辑器， 则里面已经包含了 SDK。 在 Dart 目录的 dart-sdk 子目录中。

如果你不想使用 Dart 编辑器， 你可以只下载 SDK。 SDK 支持这些系统 {% include os-choices.html %}

{% include downloads/_sdk.html %}

通过 开发通道 可以下载正在开发的 SDK，该 SDK 不稳定并且包含一些新的功能，下载地址： {% for sdk in site.custom.downloads.binaries %} {{ sdk.name }} 64-bit SDK 或者 {{ sdk.name }} 32-bit SDK. {% endfor %}

SDK 包含那些内容

SDK 目录中包含了一个 lib 目录， 该目录包含了 Dart libraries 中的库； 还有一个 bin 目录包含了一些命令行工具：

dart
独立的 Dart VM
dart2js
Dart-to-JavaScript 编译器
dartanalyzer
静态分析工具
dartdoc
API 文档生成工具
pub
Dart 包管理器

更多 SDK 的信息，请参考 README file.

安装 SDK

解压下载的 SDK 文件即可。 如果要方便的使用这些工具， 可能还需要设置一些环境变量。

设置环境变量

一些脚本依赖于一个 DART_SDK 的环境变量。 同时，你也应该把 SDK 的 bin 目录 放到系统的 path 中。

不同的系统设置 DART_SDK 和 PATH 环境变量的 方式不一样。 下面是 Mac 和 Linux 的 Bash 设置脚本： 

DART_SDK=/path/to/dart-sdk
PATH=$PATH:$DART_SDK/bin
