Dart 的 Eclipse 插件

Eclipse 的 Dart 插件提供了 和 Dart 编辑器 类似的功能。 关于 Dart 的其他工具，参考 see Dart 工具页面。

注意： 你的 Eclipse 版本必需为 3.7 或者更高版本。

下载和安装 Dart 插件

在 Eclipse中，通过菜单 Help > Install new software... 在对话框中输入地址：

http://www.dartlang.org/eclipse/update/channels/stable/

如果你想尝试最新的功能，请 尝试下 dev 版本： http://www.dartlang.org/eclipse/update/channels/dev/

Eclipse 安装完 插件后，重启即可。

下载 Dartium

要直接运行和调试 Dart 应用，你需要 安装 Dartium (一个包含 Dart VM 的 Chromium)。 由于 Dartium 执行文件有一个月的过期限制， 所以你需要每隔一个月升级一次 Dartium、插件和 SDK。

安装和升级 Dartium： 打开 Preferences 对话框， 打开 Dart > SDK 页面。 Dartium 告诉你：

  • Dartium 当前安装在哪里
  • 到哪里安装 Dartium
  • 下载 Dartium 的网址 (dartlang.org/tools/dartium/)

升级 SDK

每次升级 Dartium 或者 Dart 插件的时候， 也应该同时升级 Dart SDK。

在 Preferences 对话框中的 Dart > SDK 页面， 点击 Upgrade SDK 按钮来升级 SDK。 该页面还显示了当前安装的 SDK 版本。

提交 Bug 和新特性

创建 Bug 或者查看已有 Bug 请访问 plugin issue list。 由于插件的大部分代码和 Dart 编辑器的一样， 所以最好也同时看看 Dart Editor issues。
