其他工具下载

你喜欢用那个 IDE 和 编辑器？

对于新手我们推荐 Dart Editor 。 在你 下载的 Dart + Editor bundle 中包含了 Dart 编辑器和其他所有创建 Dart 应用需要的工具。

如果你希望使用特定的 IDE 或者编辑器， Dart 下载界面 介绍了你需要下载 那种 Dart 资源。

IDE 插件

IntelliJ IDEA and WebStorm
Eclipse

文本编辑器插件
