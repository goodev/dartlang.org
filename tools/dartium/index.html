Dartium

带有 Dart VM 的 Chromium

Dartium 是一个带有 Dart VM 的 Chromium。 使用 Dartium 你无需把 Dart 代码编译为 JavaScript 就可以 在浏览器中测试应用。 关于开发 Dart 应用的其他工具， 参考 Dart 相关工具。

警告： Dartium 是开发者预览版，可能包含安全和稳定性问题。 不要用 Dartium 作为你的日常使用浏览器！

如何使用 Dartium

下载 Dartium

如果你下载了最新版本的 Dart 编辑器， 则里面已经包含了 Dartium。

或者你可以单独下载用于 , , or . 的最新版本的 Dartium。

Dartium (Windows) Dartium (Mac) Dartium (Linux)

注意： Dartium 执行版本每隔一个月就过期了 。 当你的 Dartium 过期后， 需要重新下载一个才能继续使用 Dartium。

安装 Dartium

只需要解压下载的压缩文件即可。 如果你想让 Dart 编辑器使用一个特定版本的 Dartium， 则把解压后的文件复制到 dart-sdk 目录或者 Dart 编辑器的安装目录来替换以前的 Chromium 即可。

提交 Bug 和新特性

用 Dartium 问题模板 来提交问题。 要查询已知问题，访问 Dartium 问题列表。
