下载和安装 Dart 非常简单！

选择您的系统平台： , , or .

有两种方式下载 Dart。

方式一： 捆绑式下载所有 Dart 工具

下载的 Dart 编辑器捆绑包中包含了 所有 Dart 语言相关的工具，可以创建、编辑、测试和编译 基于 web 的和基于 服务器端的应用。

Dart 编辑器捆绑包中包含如下内容：

  • Dart Editor - 一个功能丰富的用来编辑 、调试和运行 Dart 程序 的 IDE。
  • Dartium - Chromium 的定制版本，里面包含了一个 Dart VM。 可以用 Dartium 来测试和调试 您的 Dart Web 应用。

  • SDK - 软件开发包，里面包含 Dart 虚拟机、 Dart 库、和其他一些 命令行工具：

    • dart - 独立的 VM
    • dart2js - Dart-to-JavaScript 编译器
    • dartanalyzer - 静态分析工具
    • pub - Dart 包管理器
    • dartdoc - API 文档生成工具

当前 Dart 稳定版本： [计算中]

Dart Editor 可以自己在线更新，同时也可以更新 SDK 和 Dartium，当一个新的集成版本发布的时候，你可以选择更新。 也可以到 Preferences菜单中选择， Update，然后选择 Download updates automatically 来自动更新。

早期尝鲜者？

如果您想提前几周探索稳定版中的新的功能， 你可以下载Dev Channel(开发者通道) 中的版本： Dart Editor for Windows 64-bit 或者 Dart Editor for Windows 32-bit。

如果您想提前几周探索稳定版中的新的功能， 你可以下载Dev Channel(开发者通道) 中的版本： Dart Editor for Linux 64-bit 或者 Dart Editor for Linux 32-bit。

如果您想提前几周探索稳定版中的新的功能， 你可以下载Dev Channel(开发者通道) 中的版本： Dart Editor for Mac OS X。

下载完后，解压下载的文件。 Dart 就安装好了！

现在你可以看看 海盗徽章 新手代码教程了。

方式二： 只下载你需要的软件

也许你喜欢使用自己管用的 IDE 或者编辑器，并不需要 Dart 编辑器。

没问题！

你只需要下载 Dart SDK 就可以开始使用 Dart 了。 Dart SDK 中包含如下 内容：

  • dart - 独立的 VM
  • dart2js - Dart-to-JavaScript 编译器
  • dartanalyzer - 静态分析工具
  • pub - Dart 包管理器
  • dartdoc - API 文档生成工具

如果你在编写一个 Web 应用，则你还应该下载 Dartium。 Chromium 的定制版本，里面包含了一个 Dart VM。 可以用 Dartium 来测试和调试 您的 Dart Web 应用。

Dartium 可执行文件在 12 周后过期。 当过期后，你需要重新 下载一个 Dartium 来使用。

早期尝鲜者？

如果您想提前几周探索稳定版中的新的功能， 你可以下载Dev Channel(开发者通道) 中的版本： SDK for Windows 64-bit 或者 SDK for Windows 32-bit.

如果您想提前几周探索稳定版中的新的功能， 你可以下载Dev Channel(开发者通道) 中的版本： SDK for Mac OS X.

如果您想提前几周探索稳定版中的新的功能， 你可以下载Dev Channel(开发者通道) 中的版本： SDK for Linux 64-bit 或者 SDK for Linux 32-bit.

你还可以下载Dev Channel(开发者通道) 中的： Dartium for Windows.

你还可以下载Dev Channel(开发者通道) 中的： Dartium for Mac OS X.

你还可以下载Dev Channel(开发者通道) 中的： Dartium for Linux 64-bit dartium-linux-x64-release.zip 或者 Dartium for Linux 32-bit.

注意，如果你按照这种方式来下载 Dart， 下载的文件比较小，但是当 有新版本发布的时候， 你需要手动更新一下。

现在你可以 下载一个插件 来定制你的 IDE 或者编辑器 使其支持 Dart 语言。

Linux 用户提示

如果你使用旧版的 Ubuntu 编译 SDK， 你可能需要 把 GCC 升级到 4.6+ 版本。 参考 工具 FAQ 了解详情。
