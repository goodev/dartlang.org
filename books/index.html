Dart 相关图书

下面是关于 Dart 图书的集合。 如果你发现新的图书，请告诉我们 。

The Dart Programming Language

作者： Gilad Bracha

Dart 可以更方便的开发结构化的 web 应用，从简单的脚本到 大的系统其可扩展性都比较好。The Dart Programming Language 一书是介绍 Dart 语言的 权威书籍。Gilad Bracha 领导了 Dart 语言规范的制定工作。该书适合科技工作者、大学学生、和 专业的程序开发人员。该书有意成为 Dart 语言的参考书，并介绍了 Dart 的基本原理和设计哲学。

Dart 初学者指南

作者： David Kopec

Dart 初学者指南 是为没有 web 应用编程背景的人准备的。 Dart for Absolute Beginners enables individuals with no background in programming to create their own web apps while learning the fundamentals of software development in a cutting edge language. Easily digested chapters, while comprehensive enough to explore the whole domain, are aimed at both hobbyists and professionals alike. The reader will not only gain an insight into Dart, but also the technologies behind the web. A firm foundation is laid for further programming studies.

Learning Dart

作者： Dzenan Ridjanovic and Ivo Balbaert

Learning Dart 先提供了这个新型开源语言的概览。 本书通过手把手的游戏和商业应用示例来告诉你如何开发 web 应用， 并介绍了新型语言和 HTML5 结合的威力。 Learning Dart 从介绍 Dart 生态开始。后面两章介绍所有 你需要知道的 Dart 语法和开发原则。该书的其他部分，将使用这些内容来 学习更高级的主题。

Dart for Hipsters

作者： Chris Strom

喂！你不再限制为只能用 JavaScript 了。 Dart for Hipsters 从第一页就使用真正的代码和真正的项目 来告诉你如何快速的开发网页应用。 Dart 的目标是让 Web 编程更加简单、高效和更具威力。 而 Dart for Hipsters 一书可以让你更快的掌握 Dart 语言。 在该书中从简单的示例开始到可以分割多少库的复杂应用来介绍如何使用 Dart。 警告：在阅读过该书后，你可能具有开发结构化代码的强烈冲动。

Dart in Action

作者： Chris Buckett

Dart in Action 介绍了 Dart 语言和技术，并通过示例介绍 如何用 Dart 来替代基于浏览器应用中的 Java 和 JavaScript。 本书开始快速的介绍了 Dart 语言和工具的特性，例如 和浏览器的互动、可选的类型、类、库和并发编程等。 当掌握了核心概念后，你开始在服务器上运行 Dart 代码，并创建 "只有一个页面" 组成的 HTML5 web 应用。

Dart: Up and Running

作者： Kathy Walrath and Seth Ladd

Google 创建的 Dart 语言不仅继承了 JavaScript 的动态特性还解决了其遇到的问题。 该操作行图书可以加速你了解 Dart 语言、库和工具，以及如何快速的开发可维护的结构化 web 应用。 Dart 平台被设计为既支持简单的脚本应用也支持复杂的应用，同时也支持服务器和客户端。 该书介绍了如何用 Dart 来开发现代 Web 的 HTML5 应用。

What is Dart?

作者： Kathy Walrath and Seth Ladd

这个来自于 2012 年 3 月的短小精悍的免费电子书，介绍了 describes the Dart language, Dart 语言、库和帮助你快速开发结构化的可维护的 web 应用的工具。

一种用于创建可扩展 web 应用程序的语言、工具和代码库

Dart 是一个 开源项目，由 Google 和其他人员参与。
